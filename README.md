# wordslab-voice - Use AI to transcribe speech to text and text to speech

## Live speech transcription and translation

1. Open the notebook:

> 01_speech-transcription-translation.ipynb

2. Make sure the kernel 'wordslab-voice' is selected, then execute all cells from the notebook and follow the instructions

3. Launch the web application at the end of the notebook, speak english in your microphone, and watch a live english transcription and french translation

## Audio files transcription

1. Open the notebook:

> 02_audio-files-transcription.ipynb

2. Make sure the kernel 'wordslab-voice' is selected, then execute all cells from the notebook and follow the instructions

3. Transcribe a list of audio files and reformat the extracted text to make it ready for publication

## Speech to speech conversation with your Large Language Model

1. Open the notebook:

> 03_speech-to-speech-kyutai-unmute.ipynb

2. Make sure the kernel 'wordslab-voice' is selected, then execute all cells from the notebook and follow the instructions

3. Launch the Kyutai Unmute system with docker compose, connect to the frontend web app, and speak with the model
